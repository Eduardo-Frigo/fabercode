# Knowledge Sources

This folder is intentionally public-safe.

Private Cortex bootstrap material should stay outside Git, under `private_context/` or another local-only location.

When adding public examples here, use placeholders for domains, project IDs, API keys, customer names and deployment accounts.

Do not place active Cortex memories, RAG indexes, MemPalace stores, customer workspaces or private project decisions in this folder. Public examples should explain generic behavior only.
